### Operadores relacionais

Os operadores relacionais são utilizados para comparar valores e retornar um valor lógico. Os operadores relacionais são:

| Operador | C++ | Exemplo | Significado    |
| -------- | --- | ------- | -------------- |
| ==       | ==  | 5 == 5  | Igual          |
| !=       | !=  | 5 != 5  | Diferente      |
| <        | <   | 5 < 5   | Menor          |
| >        | >   | 5 > 5   | Maior          |
| <=       | <=  | 5 <= 5  | Menor ou igual |
| >=       | >=  | 5 >= 5  | Maior ou igual |
